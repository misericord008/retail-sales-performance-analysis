# retail-sales-performance-analysis
SQL-based analysis of retail sales performance and growth opportunities
## Dataset
The dataset contains retail sales transactions across regions, product categories,
and time periods. Each record represents a single order with calculated total sales.

## Analysis Overview
The analysis focuses on answering key business questions:
- Which regions generate the highest revenue
- How product categories perform across regions
- Monthly sales trends and seasonality
- Identification of growth opportunities

## Project Structure
- `data/` – Raw dataset used for analysis
- `sql/` – SQL queries answering business questions
- `visuals/` – Charts and dashboard screenshots (optional)

## Key Outcome
This analysis highlights top-performing regions and categories while identifying
underperforming areas that present opportunities for strategic improvement.
